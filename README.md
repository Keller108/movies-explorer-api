# Backend для дипломного проекта - Movies Explorer.

### Инструменты использованы

* MongoDB
* Mongoose
* Node.js
* Express.js
* NPM
* Nginx
* Ubuntu
* Terminal
* GitHub

**Описание проекта**
Основная задача заключается в создании API. Реализована регистрация, авторизация, аутентификация, а так же поиск фильмов по БД.
